搞趣网iOS客户端广告SDK，对接第三方平台和自家平台的广告，包括横幅广告，插屏广告，开屏广告。

## 最少代码接入多个平台广告

- 线程自动执行，根据广告后台配置展示广告，并自动上报广告展示和点击；
- 支持的广告平台包括：‘广点通’、‘百度移动’、‘Inmobi’、‘Admob’、‘Gao7’;
- 支持广告类型包括：横幅广告、插屏广告、开屏广告；
